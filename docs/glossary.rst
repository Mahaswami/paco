.. _glossary:

Glossary
========

.. glossary::
   :sorted:

   AIM
     Application Infrastructure Manager (AIM).

   AIM Configuration
     Structured directory of files and sub-directories that declaratively describe an IaC project.

   AIM Schema
     A set of fields that describe the fields name, type and constraints.

   Container
     A key/value mapping where each value is a key/value mapping that corresponds to
     the AIM schema that the container is intended to hold.

   Infrastructure-as-Code
     Any software project which is intended to automatically provision and manage cloud resources.

   Waterbear Cloud
    A Software-as-a-Service (SaaS) provider that created and offers cloud orchestration services using AIM.
